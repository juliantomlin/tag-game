# phaser-express-boilerplate

# Get Started

> Install dependencies.

```terminal
> npm install
```

> Express serves static HTML, CSS and JS files.

```terminal
> npm start
```

or manually

```terminal
> node index.js
```

# Preview

> Now open http://localhost:3000 There's black canvas and a single doggy sprite. That's all.
